Règles

Dans ce jeu, vous affrontez un ordinateur au Shifumi (ou Pierre Feuille Ciseaux en français). Voici les règles:

- Un compte à rebours de 3s se lance où vous avez la possibilité de choisir entre pierre feuille ciseaux. Il suffit de cliquer sur le choix que vous souhaitez pendant ce laps de temps pour faire votre choix.

- Si vous n'avez pas choisi, le jeu va choisir aléatoirement pour vous.

- Pierre > Ciseaux > Feuille > Pierre.

- Le premier à 3 points a gagné la partie.


Si vous vous êtes connecté, vous pouvez accéder à votre profil où vous pourrez voir vos 3 dernières parties. Bon jeu !
